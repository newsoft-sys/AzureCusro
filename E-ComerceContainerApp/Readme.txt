Configuração da API no Azure
Crie uma API no Azure Functions para receber os dados do formulário.
Configure a API para salvar os dados no Azure Blob Storage ou Table Storage.
Substitua https://<sua-api-azure-endpoint>/savePost no código JavaScript pelo endpoint da sua API.
Essa página permite criar posts e enviá-los para o Azure Storage por meio de uma API. Certifique-se de configurar corretamente a API no Azure para lidar com os dados recebidos.
